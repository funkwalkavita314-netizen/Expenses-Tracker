# Expense Tracker (CLI) – Python
A simple Command Line Expense Tracker built using Python.
This program allows users to record their daily expenses, store them in a file, and analyze their spending with total and category-wise calculations.

# Features
1. Add new expenses with category and amount
2. Automatically save expenses to a file
3. View all recorded expenses
4. Calculate total expenses
5. Display category-wise spending
6. Data is saved permanently using file handling

# Technologies Used
1. Python
2. File Handling (read, write, append)
3. Lists and Dictionaries
4. Loops and Conditional Statements

# How the Program Works
1. The program loads existing expenses from a file if it already exists.
2. Users can add a new expense by entering:
-Category (Food, Travel, Shopping, etc.)
-Amount spent
3. The expense is saved to a file so it remains available even after the program closes.
4. Users can:
-View all expenses
-Calculate the total amount spent
-See category-wise totals
5. The program runs in a menu-based CLI system.

# How to Run
1. Download or clone the repository
2. Navigate to the project folder
3. Run the program
python expense_tracker.py
